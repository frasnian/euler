Euler Zero - Documentation Home

This project contains solutions to Project Euler problems in C++, with a goal of zero-runtime execution speed for each one. See the project background and introduction page for more about this.

There is a wiki for this project at https://github.com/frasnian/euler-zero/wiki - it has details about the repo, project log, which docs are where, etc. (most docs are here in the local doc directory, but there may be some additional info at the wiki).

Although there is a project wiki, the primary project documentation is included as HTML files in the repository (i.e. you are reading them). The docs can be customized a bit - the most useful things are probably setting a preferred compiler and changing default compiler options used by the individual solution readme files to build generate command lines. Details are here.

If you find bugs, have any comments, suggestions about how the documentation can be improved, etc., please drop me an email.

The Project Euler home page is at https://projecteuler.net/. I am not affiliated in any way with them, but credit for the original problems is theirs.
