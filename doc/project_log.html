Euler Zero - Project Log

There is a Project Log page on the github project wiki here.

The project log has notes about changes to the project as a whole - reorgs, structural changes, etc. It's just a place for me to document (mostly for myself) changes or decisions that affect the entire project (and why I made them). Normal changes to files within the repo itself will obviously not be logged here -- that's what git commits and revision histories are for :)

It's useful when checking items off of my TODO list.

It will probably bore you.
